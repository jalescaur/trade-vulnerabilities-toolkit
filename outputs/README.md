# outputs

This folder contains **chapter/article-ready** outputs.

## outputs/tables
Final tables that appear in the chapter, typically in Excel format:
- `outputs/tables/*.xlsx`

Optionally, provide a CSV companion for transparency:
- `outputs/tables/*.csv`

## outputs/figures
Final figures that appear in the chapter:
- PDF (vector): best for print and manuscript insertion
- PNG (600dpi): best for web preview or some submission systems
